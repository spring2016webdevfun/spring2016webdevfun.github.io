Common CSS

color h1 { color: blue; }
Text color
background-color p { background-color: blue; }
Background color
font-family - body { font-family: Arial, sans-serif; }
Font family such as Arial, serif, sans-serif
font-size - p { font-size: 2em; }
Size of the font in px, em, percentage or text values(xx-small, x-small, small, medium, large, x-large, xx-large).
margin - div { margin: 1em 1em 2em 2em; }
Margin surrounding an element. Alternate: margin-left, margin-right, margin-top, margin-bottom. px, em, percentage
padding - div { padding: 2em; }
padding-left, padding-right, padding-top, padding-bottom - h3 { padding-bottom: 2em; }
Blank space between the element and its border
text-align - div { text-align: center; }
Text alignment: center, justify, left or right
width - p { width: 80%; }
Width of a content element
text-decoration - a { text-decoration: none; }
Determines if text is underlined. none on an anchor will mean that it is not underlined
border - h1 { border: 1px dashed blue; }
border-bottom, border-top, border-left, border-right - h2 { border-bottom: 1px solid black; }
Shorthand to configure a border around an element. Width, border-style, and border color
border-radius - p { border-radius: 15px; }
Configure rounded corners
list-style-type - ul { list-style-type: none; }
Style of a list marker
display - li { display: inline;}
float - div { float: left;}
Float an element on the left or the right side of the browser window or an element. Other elements flow around the element
Clear the float by putting clear on the block element following - main { clear: both; }

Header Text Image Replacement
Replace a header with an image while making the text available to screen readers
                            header {
                                height: 2em;
                                background-image: url('customimage.jpg');
                                background-repeat: no-repeat;
                            }
                            
                            header h1 {
                                text-indent: 100%;
                                white-space: nowrap;
                                overflow: hidden;
                            }
